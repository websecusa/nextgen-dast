What this is

LLM endpoints power the Analyze with LLM action on captured flows and scans. The model receives the request, the response, and the existing regex findings, and returns a JSON list of issues with severity, evidence, and remediation.

Two backends are supported:

  • anthropic — Anthropic Messages API. Set model to your provider's current model identifier (see your Anthropic console); leave base URL blank.
  • openai_compat — anything speaking /v1/chat/completions: open-webui, LibreChat, OpenAI, Ollama, LiteLLM. Set base URL to the /v1 root (e.g. https://your-openwebui/api or https://api.openai.com/v1).
