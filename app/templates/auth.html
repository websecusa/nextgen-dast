Why this exists

Authenticated scanning needs a session. For plain logins, give us the credentials. For SSO / OAuth / SAML / Okta FastPass / DUO, those flows can't be driven headlessly — the MFA challenge is the whole point. Instead:

  1. Start the intercept proxy.
  2. Point your browser at the proxy URL and log in normally — complete the MFA challenge.
  3. Open Flows, find a request from after login, and click Save as auth profile. The system extracts your session cookies (and any Authorization header) into a reusable "cookies" profile.
  4. Pick that profile when launching a scan.

Note: nikto's CLI only supports HTTP basic auth. wapiti supports basic, form, cookies, and bearer.
