Theme

Severity colors

Used for the severity word and for the heat-map cells in the executive summary.

Brand accents

Cover page text

Foreground color for the cover. Pair with the cover background (Primary, above) — pick a contrasting value or your title text will disappear into the page.

Body text

Header color drives every h1 / h2 / h3 and the rule under each h2. Body color drives paragraph text and the inline severity labels in finding sections (severity is conveyed by border-left and heat-map cells, not by label color, so labels read like normal text).

Hyperlink color is decoupled from Primary on purpose — primary doubles as the cover-page background, so a pale brand primary would render link text invisible. Default is a print-friendly blue (#0645ad) that stays legible on a black-and-white printout.
