What this is

The User-Agent string sent by every HTTP scanner during a scan or assessment. Pick one for stealth (look like a real browser), for consistency (so you can grep your access logs), or for compatibility (some apps gate behavior on UA). The default is used when the form leaves User-Agent set to default.

Note: testssl performs raw TLS handshakes with no HTTP layer, so User-Agent doesn't apply there.
